# Hungry Elephant

A fun game where you control an elphant to eat as many falling apples as possible. 

# Authors
- Kolby
- Charis 

# Description
Help the hungry elephant eat apples falling from the sky. As you eat more apples, the games gets harder! The apples will start falling faster everytime you score 5 points. 

# How to Play
- Use the left and right arrow keys to move the elephant left & right.
- "Catch" the falling apples to score points
- Avoid the falling rocks to prevent losing points.
